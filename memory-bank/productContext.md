# Product Context: AI Tax Engine

## Why This Project Exists

Tax preparation is manual, error-prone, and expensive. Small businesses and freelancers spend hours categorizing transactions and filling out forms. Existing solutions (TurboTax, QuickBooks) require significant manual input and don't leverage AI effectively.

## Problems Solved

1. **Manual Data Entry**: AI extracts data from any format (CSV, XLSX, PDF)
2. **Transaction Classification**: AI categorizes expenses automatically
3. **Tax Form Complexity**: System generates correct forms based on entity type
4. **Audit Risk**: Full audit trail and confidence scoring
5. **Cost**: Reduce accountant dependency for routine tax prep

## How It Works

1. User creates an entity (business) with tax classification
2. User uploads financial data (bank statements, spreadsheets)
3. AI extracts and classifies transactions
4. System calculates tax liability
5. System generates PDF and XML forms
6. User reviews and approves

## User Experience Goals

- **Fast**: Import and classify in minutes, not hours
- **Accurate**: AI confidence scoring with human review for edge cases
- **Transparent**: Clear audit trail for every transaction
- **Simple**: Minimal accounting knowledge required
- **Flexible**: Support multiple entity types and tax forms

## Target Users

- Freelancers and sole proprietors
- Small business owners (LLCs, S-Corps)
- Tax preparers serving multiple clients
- Startups with simple accounting needs
